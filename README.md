# 공간정보 기본법 연구 - 국외 플랫폼 생태계 현황 분석

**담당**: 신진 (SJ), 정민우  
**지도교수**: 박교수님

## 연구 목적

공간정보 기본법 법률 개정 공포 및 본격 시행에 대비하여, 국외 지도 플랫폼(Google Maps, Apple Maps, A-Map, Yandex Maps)의 생태계 현황을 분석하고 온쇼어 보안체계 관련 시사점을 도출한다.

## 폴더 구조

```
├── 00_원본자료/          # 교수님 제공 자료, 원본 문서
├── 01_0차_완료/          # 0차 작업물 (완료)
├── 02_1차_진행중/        # 1차 작업물 (진행중)
├── 03_2차_예정/          # 2차 작업물 (예정)
├── 04_3차_예정/          # 3차 작업물 (예정)
├── 보고서_모음/           # 최종 통합 보고서 모음
└── 공간정보기본법_연구스케쥴.ipynb  # 일정 및 진행 현황 관리
```

## 진행 현황

| 차수 | 기한 | 상태 |
|------|------|------|
| 0차 | 2026-04-01 | ✅ 완료 |
| 1차 | 2026-04-08 | 🔄 진행중 |
| 2차 | 미정 | ⏳ 예정 |
| 3차 | 미정 | ⏳ 예정 |
